# Arezzo - Front-end Challenger [![Generic badge](https://img.shields.io/badge/React-JS-<COLOR>.svg)](https://shields.io/)

Neste projeto utilizei:

- ReactJS
- Styled-components
- CommitLint
- Commitizen

## Uso
Para ligar o servidor:

`yarn start`

Para fazer o build:

`yarn build`

Para commitar:

`yarn commit`
